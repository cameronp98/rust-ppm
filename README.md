# rust-ppm [![Build Status](https://travis-ci.com/cameronp98/rust-ppm.svg?branch=master)](https://travis-ci.com/cameronp98/rust-ppm)
> Rust crate for handling Netpbm images modeled on `image` v0.22.0

To run tests:
```$ cargo test```
